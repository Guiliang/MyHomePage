Research

Currently, my research is focusing on Deep Reinforcement Learning (DRL) and its application on Sport Analytic.
  • A DRL- based Player Evaluation system.
    • My first work combined the DRL model with Dynamic Trace length LSTM to model the sequential data in sports. We have built a DRL-based player ranking system and developed a novel metric to estimated the value of players, using the play-by-play Video Tracking dataset provided by sportlogiq. This work is implemented by Tensorflow with source code and accepted by main track paper in IJCAI 2018.
  • Interpret DRL model with Mimic Learning.
    • We proposed a Linear Model U-Tree (LMUT) algorithm to distill the knowledge from the opaque deep model to a transparent model with tree structure representation. The rules and knowledge are extracted from LMUT to interpret the knowledge learnt by DRL model. Paper "Toward Interpretable Deep Reinforcement Learning with Linear Model U-Trees" is submitted to the ECML 2018.
  • Combing Reinforcement Learning with Sports Demonstration.
    • A various of works has attempted to combine DRL with Imitation learning. Before interacting with environment, the agents are firstly pre-trained with the demonstration data from human experts. We are building a model with demonstrate data from the Professional Ice Hockey players in National Hockey League.
